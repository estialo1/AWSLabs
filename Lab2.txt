Step2  (Create a Key Pair)
Login into your AWS Console
Create a Key pair with name "Linux" and key file format ".pem"
Download pem file into your local machine
Create a Security Group with name “PublicSG” and give it any description you prefer
Select your default VPC
With the following  Inbound rule
Type      Source
SSH         My IP
HTTP       Anywhere IPv4
Create Launch template , give it a name and description you prefer.
Select “Amazon Linux 2 AMI “ under Application and OS Images
Select “t3micro “ for instance type
Select your pre-created Key pair you created
Select existing security group and select the one you created

Step3 (ssh into Server)
On your local machine log into the server using the keypair downloaded into your local machine
Navigate to the location of the pem key and run the following commands
chmod 400 <name of pem key>
ssh -i <name of pem key> ec2user@<public-ip-address-of server>
This gets you into the server

Step4 (Set up page and nginx service)
Run the following commands
yum update -y
yum install -y nginx
systemctl start nginx
systemctl enable nginx
echo "Hello World from $(hostname -f)" | sudo tee /usr/share/nginx/html/index.html
systemctl restart nginx
On a fresh web page paste http://<public ip address of your Virtual machine>
It should show Hello World from <public ip address of your Virtual machine>
Share this http://<public ip address of your Virtual machine> on the WhatsApp group for confirmation
Once confirmed, terminate the instance on you AWS Console

Setting Up Nginx Webpage on EC2

Step 1 — SSH Into the EC2 Instance**

1. Downloaded the key pair `Linux.pem` from AWS when creating the EC2 key pair.
2. Opened WSL terminal and navigated to the location of the key:

cd /mnt/c/Users/phantom/Downloads

3. Copied the key into the WSL home directory


cp Linux.pem ~/
cd ~

4. Set proper permissions on the key:

chmod 400 Linux.pem

5. Connected to the EC2 instance using SSH:

ssh -i Linux.pem ec2-user@<Public-IP>

Once logged into the EC2 instance:

sudo yum update -y to update the server

sudo yum install -y nginx to install nginx

sudo systemctl start nginx to start the nginx server

sudo systemctl enable nginx to Enable nginx to start on boot

sudo systemctl status nginx to Verify nginx is running
**


echo "Hello World from $(hostname -f)" | sudo tee /usr/share/nginx/html/index.html to Create a simple webpage for testing
`$(hostname -f)` automatically inserts the EC2 internal hostname.

echo "Hello World from Esther" | sudo tee /usr/share/nginx/html/index.html to show my name

sudo systemctl restart nginx  Restarted nginx to apply changes

curl http://localhost to test the webpage

Output should show:

Hello World from ip-172-31-xx-xx.ec2.internal
Hello World from Esther
